# NutriAI 换网络重启速查

## 每次换热点 / 换 Wi-Fi 后的步骤

### ① 查电脑新 IP

```bash
ipconfig
```

找到 **WLAN** 的 **IPv4 地址**，记下来，比如 `192.168.x.x`。

### ② 改 `.env`（backend/.env）

```
MEDIA_BASE_URL=http://192.168.x.x:8000
```

### ③ 重启 Docker

```bash
cd F:\课设\大数据可视化\设计六\backend
docker-compose down
$env:GEMINI_API_KEY="你的key"; docker-compose up -d
```

### ④ 更新数据库旧图片地址

```bash
docker exec backend-db-1 psql -U postgres -d nutriai -c "UPDATE media_assets SET image_url = REPLACE(image_url, 'http://旧IP:8000', 'http://192.168.x.x:8000');"
docker exec backend-db-1 psql -U postgres -d nutriai -c "UPDATE meals SET image_url = REPLACE(image_url, 'http://旧IP:8000', 'http://192.168.x.x:8000');"
```

### ⑤ 改前端 ApiClient.kt + 重装 APK

`ApiClient.kt`：

```kotlin
private const val BASE_URL = "http://192.168.x.x:8000/api/v1/"
```

构建安装：

```bash
cd F:\code\Program\new\frontend
gradlew assembleDebug
adb install -r app\build\outputs\apk\debug\app-debug.apk
```

## 如果插 USB 用 adb reverse（不需要改 IP）

```bash
# 只需这一行，不用改任何配置
adb reverse tcp:8000 tcp:8000

# 构建安装 APK（BASE_URL 保持 http://localhost:8000/api/v1/）
cd F:\code\Program\new\frontend
gradlew assembleDebug
adb install -r app\build\outputs\apk\debug\app-debug.apk
```
