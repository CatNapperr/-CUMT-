# NutriAI 真机调试速查

## 前置条件

```
电脑端：后端正在运行（docker-compose up -d 或 uvicorn）
手机端：USB 连接电脑，开启「开发者选项」和「USB 调试」
```

---

## 步骤

### ① 端口映射（每次重插 USB 都要做）

```bash
adb reverse tcp:8000 tcp:8000
```

### ② 确保前端 BASE_URL 用 localhost

`ApiClient.kt`：

```kotlin
private const val BASE_URL = "http://localhost:8000/api/v1/"
```

### ③ 确保后端 MEDIA_BASE_URL 用 localhost

`.env`：

```ini
MEDIA_BASE_URL=http://localhost:8000
```

### ④ 构建 & 安装 APK

```bash
cd F:\code\Program\new\frontend
gradlew assembleDebug
adb install -r app\build\outputs\apk\debug\app-debug.apk
```

### ⑤ 打开 App

手机上的 NutriAI 会自动通过 USB 映射连接到电脑的后端。

---

## 拔线后用 Wi-Fi

```bash
# 查电脑 IP
ipconfig
# 找到 WLAN 的 IPv4 地址，如 10.4.215.252
```

改三处：

| 文件 | 改为什么 |
|---|---|
| `ApiClient.kt` | `http://10.4.215.252:8000/api/v1/` |
| `.env` | `MEDIA_BASE_URL=http://10.4.215.252:8000` |

重新构建 APK 安装。手机和电脑需连同一 Wi-Fi。

---

## 后端启动命令

```bash
# Docker 方式
cd F:\课设\大数据可视化\设计六\backend
$env:GEMINI_API_KEY="你的key"; docker-compose up -d

# 或直接跑（不用 Docker）
cd F:\课设\大数据可视化\设计六\backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```
